Macro to disable the Internal 14Mhz High Speed oscillator (HSI14). 

\begin{DoxyNote}{Note}
The HSI14 is stopped by hardware when entering STOP and STANDBY modes. 

HSI14 can not be stopped if it is used as system clock source. In this case, you have to select another source of the system clock then stop the HSI14. 

When the HSI14 is stopped, HSI14\+RDY flag goes low after 6 HSI14 oscillator clock cycles. 
\end{DoxyNote}
